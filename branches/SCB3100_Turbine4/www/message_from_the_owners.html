The Scarab 1.0.22 Release Candidate 1 is available - please see the Announcement for full details. There are three alternatives for trying scarab:

  • Use the embedded build (full out-of-the-box experience), details can be found in the Announcement and the zip archive can be found in the Release Candidate folder of the Document & files section.
  • You can try the new web based build system here.
  • Alternatively, as usual, the source files can be found in the Release Candidate folder of the Document & files section.

As the release candidate is already extensively tested we hope to have a final release soon, but of course we appreciate your help with testing, feedback and bug fixing. If you like to help us, you always can checkout the newest sources from our subversion repository at http://project.tigris.org/svn/scarab/branches/release/1.0.22 / and report to the Mailing List(s) or the Scarab issue tracker
